Checkbox & radio Elements

  • 기본적용은 레이블태그와 같이 적용
  • checkbox, radio 디자인 적용은 input의 type으로 결정
  • 2줄이상은 멀티클래스적용(.multi-line) 정렬은 상단으로 변경
  • 레이블이 없이 단독사용시는 title속성 부여
  • 임시수치로 디자인 적용후 수정예정 .chk-group으로 랩핑시 .chk의 기본 마진은 right:20px
  • 텍스트노출없이 사용할 경우라도 빈 label(디자인이 적용되는 부분)태그는 필요

  • 밸리데이션의 속성은 data-state로 설정하며, 기본은 속성만 선언하고 value는 미기입
  • 밸리데이션 성공, 실패는 data-state=success|fail
  • 파일필드는 label에 포커스를 주며, 선택시 빈 input에 텍스트를 전달
  • 파일필드의 사이즈(.ipt-file)는 기본 블럭화이며, 블럭사이즈가 아닌 경우는 인스타일로 정의

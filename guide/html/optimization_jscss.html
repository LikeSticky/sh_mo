jquery/css

  • css, js의 경우 수정이 발생하지 않는 파일들은 압축형태로 제공(*.min.js|css)
  • 압축형태로 제공되는 파일중에 core를 제외한 파일들은 원본파일을 보관(*.[source].js|css)하여 추후수정에 사용
  • 압축파일의 용량은 아래와 같으며, 속도 여러상황에 따라 다르지만 통상적으로 25%-35% 개선효과(용량이 적은 파일일수록 압축률은 낮아질수 있음)를 기대할수 있음.

제공된 파일로 사용


code pattern : public

                        <link href="/css/ini.min.css" rel="stylesheet"> 
                        <link href="/css/module.css" rel="stylesheet">
                        <link href="/css/contents.css" rel="stylesheet">
                        <script type="text/javascript" src="/js/jquery-2.2.4.min.js"> 
                        <script type="text/javascript" src="/js/ui.js">

code pattern : private

                        <script type="text/javascript" src="/js/jquery.bxslider-rahisified.min.js"> 
                        <script type="text/javascript" src="/js/google.map.js">

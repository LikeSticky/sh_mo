웹접근성 검증툴 체크는 하였는가?
한국형 웹 콘텐츠 접근성 지침에 따라 세부지침을 준수하였는지를 자동으로 점검해주는 검증을 하였는가?
자동평가 툴은 많이 있겠으나 국내에서 많이 사용중인 아래 툴중 1가지 이상에서 체크를 한다.
  • 웹접근성 평가도구 소개
  • 한국형 웹 콘텐츠 접근성 지침 2.1 가이드라인 내려받기
HTML validator 통과했는가?
HTML 문법오류체크 및 Entity 문자의 변환이 잘 되었는지 확인.
  • W3C의 Markup Validation Service
CSS validator 통과했는가?
CSS문법오류체크. 특정브라우저에서만 지원하는 CSS속성 및 CSS3속성을 사용함으로 체크되는 오류는 체크사항에서 제외됨.
  • W3C의 CSS Validation Service
HTML5 마크업 체크는 하였는가?
HTML5에서 추가되는 tag나 사용하지 않는 tag등을 충분히 숙지후 Markup을 진행한다.HTML5 한글 명셋 사이트, W3Schools
  • HTML5 명세서 사이트
  • W3Schools
Cross Browsing 되는가?
모든 브라우저에서 동등한 콘텐츠를 제공 하고 있는지 논리적 구조에 맞게 콘텐츠 제공이 되고 있는지(Layout) 체크.
  • Window : Edge, Firefox, Chrome, Whale Safari,
  • MAC : Safari

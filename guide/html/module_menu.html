tab menu

  • 탭의 기능인 data-role="tab", 탭의 스타일은 클래스로 정의하여 분리
  • 탭의 메뉴영역인 area-tab-mnu와 탭의 컨텐츠 영역인 area-tab-cont를 명시선언
  • 각각의 role은 스크립트에서 처리
  • 메뉴의 aria-controls과 컨텐츠의 id연결은 스크립로 랜덤문자를 생성
  • 페이지별 단건인 테이블에서 SHS.tab.active("[data-role='tab']", 0) 인덱스번호로 탭의 활성화를 변경
  • 탭의 균등사이즈는 탭영역인 li의 flex속성을 1로 정의하고, 앵커부분의 word-break(1byte문자) 확인
  • .tabs.type01은 임시적용된 스타일
  • 페이지내 여러개의 탭메뉴와 탭안의 멀티탭 테스트필요
